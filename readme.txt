== Description ==

**Email Recipients for WooCommerce** is a lightweight plugin that lets you set custom recipients for any WooCommerce email.

### ðŸš€ Customizable Email Recipient Lists ###

Configure unique recipient lists for different types of WooCommerce emails. You can specify a comma-separated list of emails, including placeholders like `%customer%`, `%admin%`, and `%default%`, for precise targeting. Ideal for businesses wanting to direct specific notifications to different departments or team members.

== Installation ==

1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start by visiting plugin settings at "WooCommerce > Settings > Email Recipients".
